Local AI Assistant

Projeto em Python que conversa com um modelo local via LM Studio e executa ações simples no Windows.

## Como rodar

### 1. Criar a virtual environment

```powershell
python -m venv .venv
```

### 2. Ativar a virtual environment

```powershell
.\.venv\Scripts\Activate.ps1
```

### 3. Instalar dependências

```powershell
pip install -r requirements.txt
```

### 4. Executar o projeto

```powershell
python main.py
```

Ou abrir a interface:

```powershell
python main.py --gui
```
